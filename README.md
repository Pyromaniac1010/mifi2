# MiFi — Personal Finance & Debt Freedom Platform

A mobile-first PWA to track income (active vs passive), crush debt with the
Avalanche Method, watch a Financial Solvency Score, and get advice from "Mi",
a rule-based assistant with four switchable personalities.

## Stack
React 18 + Vite, Tailwind CSS, Firebase (Auth + Firestore with offline cache).

## Local development
1. npm install
2. Copy .env.example to .env and fill in your Firebase web config values.
3. npm run dev

## Deploy
Hosted on Vercel; data and auth on Firebase.
